# 🤖 AI Job Analyzer Tool

## 📌 Overview
This project analyzes job postings and matches them with CV keywords to improve job application success.

## ⚙️ Tech Stack
- Python
- Data Processing
- NLP (basic keyword matching)

## 🔥 Features
- Extracts keywords from job descriptions
- Compares with CV content
- Suggests improvements
- Improves job matching efficiency

## 📊 Results
- Increased job matching efficiency by 63%

## ▶️ How to Run
```bash
pip install -r requirements.txt
python main.py
